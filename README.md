# üì° RigRadar AI

**RigRadar AI** is an autonomous hardware market scanner and AI-powered PC building assistant. It monitors hardware prices across multiple platforms (eBay, Amazon, Alza, Bazos) 24/7, providing users with real-time deal alerts, market analytics, and intelligent hardware consultations. 

Built with serverless architecture and powered by Google's Gemini 2.0 Flash, RigRadar is your ultimate tool for finding the best PC parts at the lowest prices.

## ‚ú® Core Features

### üîç Autonomous Market Scanner (Radar Control)
* **Live Monitoring:** Continuously intercepts hardware signals from various stores to find the best deals in real-time.
* **Custom Satellites (Watchdogs):** Deploy background trackers targeting specific hardware (e.g., "RTX 4080 Super") with price thresholds and condition filters.
* **Real-time Updates:** Integrated with Pusher for instant UI updates the moment a new deal drops.

* **Integrated Scanning:** The AI can directly suggest exact search terms to plug into the Radar Control.

### üìä Market Analytics & Predictive Tools

* **Predictive Forecaster:** Provides market signals ("Buy Now," "Wait/Hold," or "Overpriced") based on historical data.
* **Price History Charts:** Visualizes market trends using Chart.js to help you time your purchases perfectly.
* **Hardware Battle:** Compare two components head-to-head to see which offers better performance and value for your specific use case.

* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript (ES6+), Chart.js

## üöÄ Installation & Setup

### Prerequisites
* Node.js (v18+)
* Vercel CLI (optional, for local serverless development)
* Upstash Redis account
* Google Gemini API Key
* Stripe Account
* Pusher Account
* Resend Account (for emails)

### Environment Variables
Create a `.env` file in the root directory and add the following keys:

```env
# Upstash Redis
KV_REST_API_URL=your_upstash_url
KV_REST_API_TOKEN=your_upstash_token

# Google Gemini AI
GEMINI_API_KEY=your_gemini_key

# Authentication
GOOGLE_CLIENT_ID=your_google_client_id
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret

# Stripe Payments
STRIPE_SECRET_KEY=your_stripe_secret
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable
STRIPE_WEBHOOK_SECRET=your_stripe_webhook_secret

# Pusher (Real-time events)
PUSHER_KEY=your_pusher_key
NEXT_PUBLIC_PUSHER_KEY=your_pusher_key

# Resend (Emails)
RESEND_API_KEY=your_resend_key

# Security & Cron
CRON_SECRET=your_secure_cron_secret
RADAR_API_SECRET=your_radar_node_secret

```

### Running Locally

1. Clone the repository:
```bash
git clone [https://github.com/yourusername/rigradar.git](https://github.com/yourusername/rigradar.git)
cd rigradar

```


2. Install dependencies:
```bash
npm install

```


3. Build Tailwind CSS:
```bash
npm run build

```


4. Start the local Vercel dev server:
```bash
vercel dev

```

## üîí Security

* **HttpOnly Cookies:** Sessions are managed via secure `HttpOnly` cookies to prevent XSS-based token theft.
* **Rate Limiting:** Anti-spam protection implemented for support and AI chat functions using Redis counters.
* **Data Sanitization:** All search queries and messages are sanitized to prevent prompt injection and XSS attacks.
* **Password Hashing:** Native Node.js `crypto.scryptSync` used with unique salts for local email/password authentication.
